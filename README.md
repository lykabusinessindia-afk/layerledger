# LayerLedger

LayerLedger is a smart 3D printing cost calculator designed to help creators and small businesses calculate accurate pricing.

## 🚀 Features
- Calculate material cost
- Estimate print time cost
- Include electricity & operational costs
- Reduce guesswork in pricing

## 🎯 Goal
To simplify pricing for 3D printing businesses and integrate seamlessly with Shopify.

## 🛠 Tech Stack
- Next.js
- TypeScript
- React
- Supabase
